# Управление категориями
На этой странице создаются и удаляются категории блюд.

::: danger Рекурсивное удаление блюд
При удалении категории удаляются все блюда, находящиеся в ней. Редактировать название категории нельзя, поэтому важно проверить его корректность при создании категории.
:::

::: danger Неожиданное поведение
Рекомендуется отправлять форму через кнопку, встроенную в форму. При отправке, например, нажатием клавиши "Enter" возможно неожиданное поведение.
:::
