# Какую проблему решает проект?
Проект позволяет вам в режиме реального времени видеть, что сейчас есть в наличии в столовой, а что уже закончилось.

Раньше приходилось спрашивать что‑то вроде: «Что сейчас есть из мяса?», «Какой сегодня суп?». Теперь же человек, стоящий в очереди, заранее знает, что сейчас есть в наличии и что он хочет купить.
