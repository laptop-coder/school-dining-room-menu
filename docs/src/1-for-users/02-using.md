# Как пользоваться сайтом?

Перейдите на главную страницу. Вы видите два переключателя. Первым вы выбираете доступность, а вторым — категорию отображаемых блюд.

Надпись «В наличии» на карточке блюда означает, что в данный момент оно имеется в столовой и его можно купить. «Нет в наличии» же значит, что либо оно закончилось, либо его сегодня вовсе не готовили.

::: info Обратите внимание
Для получения новых данных необходимо обновить страницу.
:::
